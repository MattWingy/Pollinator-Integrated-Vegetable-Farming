## How to Report Issues

Found an error or have a suggestion? Please:

1. Check existing [issues](https://github.com/MattWingy/Pollinator-Integrated-Vegetable-Farming/issues) to avoid duplicates
2. Create a new issue with:
   - Clear title describing the problem
   - Detailed description of the issue
   - Specific section/page references if applicable
